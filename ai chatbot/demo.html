This page is only a test harness. The chatbot itself is the floating button in the bottom-right corner. Click it to start

The "one simple connection"

To put this examiner on any page — including a rebuilt dashboard — you add exactly two lines. That's the entire integration:

<script src="ai chatbot/widget/amsa-chatbot.js"></script>
<script>
  AMSAChatbot.init({ apiBase: "http://localhost:3000/api" });
</script>

Nothing else on the page needs to change, and none of the existing dashboard code is touched.

Options you can pass to init()

AMSAChatbot.init({
  apiBase: "http://localhost:3000/api", // where the backend server is
  licenceType: "M45",   // "", "M45", "MED1", "ENG3" - filters the question bank
  topic: "",            // optional topic filter
  questionCount: 6,     // how many questions per session
  voice: true,          // browser speech-to-text + text-to-speech
  startOpen: false,     // open the panel automatically on load
  title: "AMSA Practice Examiner"
});

You can also call AMSAChatbot.open() / AMSAChatbot.close() — e.g. wire the dashboard's existing "Practice Examination" button to AMSAChatbot.open().
