This page is just a test harness. The assistant is the floating button in the bottom-right corner. Click it to chat

The "one simple connection"

To add the help assistant to any page — including a rebuilt dashboard — you add exactly two lines:

<script src="ai chatbot/widget/amsa-chatbot.js"></script>
<script>
  AMSAChatbot.init({ apiBase: "http://localhost:3000/api" });
</script>

Nothing else on the page changes, and none of the existing dashboard code is touched.

Options you can pass to init()

AMSAChatbot.init({
  apiBase: "http://localhost:3000/api", // where the backend server is
  title: "Help Assistant",
  startOpen: false                      // open automatically on load
});

You can also call AMSAChatbot.open() / AMSAChatbot.close() from your own buttons.
